## 缺口总结

- **已覆盖**：9个工具（文档管理3 + 内容编辑2 + 格式排版3 + 目录1）
- **未覆盖**：14个action（占比60.9%）
- **缺口分布**：
  - 核心编辑能力（表格/段落/分页/图片）：4个
  - 页面排版（页面设置/页眉/页脚）：3个
  - 文档导航（超链接/书签×2）：3个
  - 协作审阅（评论×2）：2个
  - 基础信息（活动文档/文档统计）：2个
- **建议新增文件归属**：
  - word/document.ts 补充2个（getActiveDocument, getDocumentStats）
  - word/content.ts 补充8个（表格/分页/页眉/页脚/超链接/书签×2/评论×2/图片）
  - word/format.ts 补充2个（setParagraph, setPageSetup）
  - 新建 word/collaboration.ts 可选（评论相关可独立，但建议先归入content.ts保持简洁）
